En la clase de hoy he aprendido sobre
la etiquetas div Este código HTML
representa la estructura básica de 
una interfaz web que permite al 
usuario interactuar con una lista
de palabras. Incluye un campo de
entrada de texto y varios botones
con funciones específicas: 
agregar, borrar, mostrar y
modificar. Estos elementos están 
organizados dentro de un contenedor
principal y acompañados por un
área de resultados donde se 
espera que se muestren los datos
procesados. El documento también 
enlaza una hoja de estilos externa y un 
archivo JavaScript que probablemente contiene
 la lógica de funcionamiento de los botones.

La presencia del atributo `type="module"` 
en el script indica que se está utilizando 
JavaScript moderno, lo que permite importar
y exportar funciones entre archivos. Aunque
el contenido de los archivos `style.css` y `app.js`
no está incluido, se puede deducir que el 
primero define el diseño visual de la página 
y el segundo gestiona la interacción del usuario 
con los elementos HTML.
